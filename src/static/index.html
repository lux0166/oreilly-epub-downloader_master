How to export cookies?
  1. Log into learning.oreilly.com in your browser.
  2. Open Developer Tools (Press F12 / Cmd+Opt+I).
  3. Go to Console and run this command: 
    JSON.stringify(Object.fromEntries(document.cookie.split('; ').map(c => c.split('='))))
  4. Copy the output and save it as cookies.json.
  5. Drag and drop the file in the area above.
